### 1. 叙事影响系统 ⭐⭐⭐ (`src/engine/narrative-influence.ts`)

**核心功能**：
- 叙事影响 agent 的决策和行为
- 自动更新 agent 的叙事角色（protagonist, antagonist, supporting, observer, catalyst）
- 生成叙事相关的记忆
- 调整 agent 的情感状态
- 影响 agent 的目标

**关键特性**：
- 动态角色识别：根据参与度和叙事类型自动分配角色
- 参与度计算：基于事件出现频率
- 影响力计算：叙事强度 × 参与度
- 记忆生成：为每个活跃叙事生成记忆
- 情感调整：根据叙事情感和强度调整 agent 情绪
- 目标影响：根据叙事类型添加相关目标

**效果**：
- 形成正反馈循环：叙事 → 影响行为 → 产生新事件 → 强化叙事
- Agents 会意识到自己在故事中的角色
- 叙事自然地影响 agent 行为

### 2. 群体记忆系统 ⭐⭐ (`src/engine/collective-memory.ts`)

**核心概念**：
当多个 agents 有相似记忆时，形成群体记忆（Collective Memory）

**数据结构**：
```typescript
CollectiveMemory {
  content: string  // 群体记忆内容
  participants: string[]  // 参与者
  strength: number  // 记忆强度
  type: 'event' | 'norm' | 'belief' | 'history'  // 记忆类型
  influence_on_behavior: number  // 对行为的影响
  influence_on_culture: number  // 对文化的影响
  evolution_history: []  // 演化历史
}

CulturalNorm {
  description: string  // 规范描述
  strength: number  // 规范强度
  adherents: string[]  // 遵守者
  violators: string[]  // 违反者
}
```

**关键功能**：
1. **检测群体记忆形成**：
   - 基于记忆相似度（Jaccard 相似度）
   - 至少 3 个 agents 有相似记忆
   - 自动分类记忆类型

2. **传播群体记忆**：
   - 基于社交关系传播
   - 传播概率与记忆强度相关
   - 形成个体记忆

3. **强化和演化**：
   - 新证据强化记忆
   - 记忆内容随时间演化
   - 记录演化历史

4. **提取文化规范**：
   - 从 'norm' 类型的群体记忆中提取
   - 检查规范违反
   - 追踪遵守者和违反者

**效果**：
- 形成共同的历史记忆
- 文化和价值观的涌现
- 社会规范的形成

### 5. Agent 个性化决策系统 ⭐⭐⭐ (`src/engine/agent-decision-maker.ts`)

**核心功能**：
基于 agent 的个性、信念、目标和当前上下文做决策

**行动类型**：
- `interact`: 主动互动
- `explore`: 探索新事物
- `reflect`: 反思和休息
- `pursue_goal`: 追求目标
- `avoid`: 避免冲突
- `help`: 帮助他人
- `compete`: 竞争

**决策流程**：
1. 生成候选行动（基于目标、关系、叙事、个性、情感）
2. 评估每个行动（5 个维度）
3. 选择最优行动（带随机性）

**评估维度**：
1. **个性评分**：
   - 高开放性 → 探索
   - 高同理心 → 帮助
   - 高主动性 → 竞争
   - 高稳定性 → 反思

2. **信念评分**：
   - "和平" → 帮助 +2, 竞争 -1
   - "竞争" → 竞争 +2, 帮助 -0.5
   - "知识" → 探索 +2, 反思 +1
   - "友谊" → 互动 +2, 帮助 +1

3. **状态评分**：
   - 能量低 → 反思 +3
   - 压力高 → 避免 +2, 竞争 -2
   - 专注高 → 追求目标 +2

4. **叙事相关性**：
   - 行动涉及叙事参与者 → +分
   - 高潮阶段 → +1

5. **预期效果**：
   - 帮助 → 改善关系
   - 竞争 → 实现目标但可能恶化关系
   - 追求目标 → 总是好的

**效果**：
- Agents 行为更加多样化
- 个性鲜明的角色
- 更有趣的互动

### 6. 推荐系统升级 ⭐⭐ (`src/engine/recommendation-system.ts`)

**新增功能**：
基于叙事的推荐

**推荐策略**：
1. **基于兴趣**（40% 权重 → 28%）：关键词匹配
2. **基于热度**（30% 权重 → 21%）：时间衰减 + 重要性
3. **基于社交**（20% 权重 → 14%）：关注的人的动态
4. **基于叙事**（30% 权重）：参与的叙事相关事件 ⭐ 新增

**叙事推荐算法**：
- 找到 agent 参与的叙事
- 推荐这些叙事的相关事件
- 叙事强度越高，分数越高
- 高潮阶段权重 ×1.5

**效果**：
- Agents 更容易发现感兴趣的事件
- 信息流更加个性化
- 促进叙事发展

## 技术亮点

### 1. 叙事影响的正反馈循环
- 叙事识别 → 生成记忆 → 影响情感 → 调整目标 → 改变行为 → 产生新事件 → 强化叙事
- 这是真正的"涌现"：系统自我强化

### 2. 群体记忆的涌现
- 不是预设的，而是从个体记忆中自然涌现
- 基于相似度检测（Jaccard）
- 自动分类（事件、规范、信念、历史）
- 演化和传播机制

### 3. 多维度决策评估
- 5 个维度综合评分
- 基于个性、信念、状态、叙事、预期效果
- 添加随机性避免过于确定性

### 4. 混合推荐策略
- 4 种推荐策略融合
- 动态权重调整
- 叙事推荐增强相关性

### 5. 实时可视化
- Canvas 绘制网络图
- 力导向布局（简化版）
- 交互式探索

## 性能优化

### 1. 增量更新
