# 下一步功能优先级清单

基于当前系统状态和架构设计，以下是值得加入的功能，按优先级排序。

## 🔥 P0 - 核心功能（立即实现）

### 1. 叙事影响系统
**为什么重要**：叙事识别出来了，但还没有反馈到 agents 的决策中

**功能**：
- Agents 感知到自己参与的叙事
- 叙事影响 agent 的情绪和决策
- 高强度叙事会吸引更多 agents 关注

**实现**：
```typescript
// src/engine/narrative-influence.ts
class NarrativeInfluenceSystem {
  // 将叙事信息注入到 agent 的上下文中
  applyNarrativeInfluence(agent: PersonalAgentState, narratives: NarrativePattern[]): PersonalAgentState
  
  // 计算叙事对 agent 的影响
  calculateNarrativeImpact(agent: PersonalAgentState, narrative: NarrativePattern): number
  
  // 生成叙事相关的记忆
  generateNarrativeMemories(agent: PersonalAgentState, narratives: NarrativePattern[]): MemoryRecord[]
}
```

**预期效果**：
- Agents 会意识到自己在故事中的角色
- 叙事会自然地影响 agent 行为
- 形成正反馈循环：叙事 → 影响行为 → 产生新事件 → 强化叙事

## 🌟 P1 - 重要功能（近期实现）

### 4. Agent 个性化决策系统
**为什么重要**：当前 agents 的决策还比较简单

**功能**：
- 基于 agent 的 occupation、core_belief、expertise 做决策
- 不同性格的 agents 对同一事件有不同反应
- 决策考虑当前参与的叙事

**实现**：
```typescript
// src/engine/agent-decision-maker.ts
class AgentDecisionMaker {
  // 基于个性和上下文做决策
  makeDecision(agent: PersonalAgentState, context: DecisionContext): AgentAction
  
  // 评估行动的预期效果
  evaluateAction(agent: PersonalAgentState, action: AgentAction): number
  
  // 选择最优行动
  selectBestAction(agent: PersonalAgentState, candidates: AgentAction[]): AgentAction
}
```

**预期效果**：
- Agents 行为更加多样化
- 个性鲜明的角色
- 更有趣的互动

### 5. 事件推荐系统升级
**为什么重要**：当前推荐系统还很基础

**功能**：
- 基于 agent 兴趣推荐事件
- 基于社交网络推荐（朋友在关注什么）
- 基于叙事推荐（参与的叙事相关事件）
- 混合推荐策略

**实现**：
```typescript
// 升级 src/engine/recommendation-system.ts
class EnhancedRecommendationSystem {
  // 基于兴趣的推荐
  recommendByInterest(agent: PersonalAgentState, events: WorldEvent[]): WorldEvent[]
  
  // 基于社交网络的推荐
  recommendBySocialNetwork(agent: PersonalAgentState, world: WorldSlice): WorldEvent[]
  
  // 基于叙事的推荐
  recommendByNarrative(agent: PersonalAgentState, narratives: NarrativePattern[]): WorldEvent[]
  
  // 混合推荐
  hybridRecommend(agent: PersonalAgentState, world: WorldSlice): WorldEvent[]
}
```

**预期效果**：
- Agents 更容易发现感兴趣的事件
- 信息流更加个性化
- 促进叙事发展

## 💭 我的建议

基于当前系统状态，我强烈推荐先实现：

1. **叙事影响系统**（最重要）
   - 这是闭环的关键：叙事 → 影响 → 行为 → 事件 → 叙事
   - 没有这个，叙事只是"观察"，不是"参与"

2. **社交网络可视化**
   - 可视化是理解复杂系统的关键
   - 帮助发现有趣的模式

3. **Agent 个性化决策**
   - 让每个 agent 真正独特
   - 提升叙事质量

这三个功能会让系统从"能识别叙事"变成"能产生有趣的叙事"。
