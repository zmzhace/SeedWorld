## 🎯 项目愿景

World Slice 不是传统的故事生成器，而是一个**复杂自适应系统**，灵感来源于：
- 🎮 **OASIS** - 虚拟世界的沉浸感
- 🐟 **MiroFish** - Agent 的自主性和涌现行为
- 📖 **StoryBox** - 叙事的自然形成

我们的目标是创造一个世界，在这里：
- ✨ **故事自然涌现**，而非预先编写
- 🤖 **Agents 真实生活**，而非执行脚本
- 🌊 **复杂性自组织**，而非人工设计
- 🎭 **戏剧性自发产生**，而非刻意制造

## 🌟 核心特性

### 🧠 智能 Agent 系统

#### 盘古（Pangu）- 世界生成
- 🌍 基于 LLM 生成初始世界状态
- 🎲 动态环境、社会背景、叙事种子
- 🔄 持续演化的世界设定

#### 女娲（Nuwa）- Agent 创造
- 👤 动态生成个性化 Agents
- 🎭 5 维个性系统（开放性、稳定性、同理心、依恋、主动性）
- 💼 LLM 生成的独特职业和信念
- 🎯 自主目标和动机

#### 后土（Houtu）- 生死轮回
- ⚰️ 自然的生命周期管理
- 🔄 轮回机制（继承特质和记忆）
- 💫 遗产系统（影响后代）

### 📖 涌现式叙事系统

#### 叙事识别器
- 🔍 自动识别 10 种叙事模式
  - 冲突、联盟、背叛、救赎、成长
  - 牺牲、发现、复仇、爱情、权力斗争
- 📊 追踪叙事发展（萌芽 → 发展 → 高潮 → 结束）
- 🎬 三幕剧结构检测

#### 故事弧检测器
- 📈 识别完整的故事弧
- 🎭 主角、反派、配角识别
- 🌊 情感曲线追踪
- ⏱️ 时间线管理

#### 叙事总结器
- 📝 多视角叙事总结
- 🎨 情感色彩分析
- 🔗 事件关联发现

### 🧬 20 个高级机制系统

#### 🎭 认知与心理（已实现）
1. **认知偏差系统** - 7 种偏差影响决策
   - 确认偏差、锚定效应、可得性启发
   - 群体思维、沉没成本、乐观偏差、损失厌恶
2. **元认知系统** - Agents 反思自己的思考
   - 自我意识、决策信心、学习速度
   - 策略调整、洞察生成
3. **注意力机制** - 有限的注意力容量
   - 选择性关注、注意力转移
   - 疲劳累积和恢复

#### 🌐 社会与文化（已实现）
4. **社会角色系统** - 4 类角色（职业、家庭、社交、文化）
   - 角色期望、义务、特权
   - 角色冲突检测和解决
5. **声誉系统** - 5 维度声誉追踪
   - 可信度、能力、善意、地位、影响力
   - 声誉传播、衰减、修复
6. **模因传播系统** - 思想病毒式传播
   - SIR 传播模型、模因变异
   - 模因竞争和选择

#### 🎬 叙事与戏剧（已实现）
7. **戏剧张力系统** - 5 种张力类型
   - 悬念、惊奇、好奇、冲突、时间压力
   - 自动累积和释放
8. **伏笔与回响系统** - 叙事深度增强
   - 伏笔种植和兑现
   - 历史回响和意义赋予
9. **多视角叙事系统** - 罗生门效应
   - 主观记忆、信息不对称
   - 解读差异、真相重构

#### 🔬 系统动力学（已实现）
10. **涌现属性检测器** - 发现系统新特性
    - 相变、临界点、自组织
    - 同步、集体行为
11. **反馈回路分析器** - 系统动力学分析
    - 正反馈、负反馈、延迟反馈
    - 回路强度和稳定性
12. **蝴蝶效应追踪器** - 因果链追踪
    - 影响放大、关键节点
    - 反事实分析

#### 🧬 进化与学习（已实现）
13. **文化进化系统** - 文化特征进化
    - 变异、选择、遗传
    - 文化适应和漂变
14. **强化学习系统** - 试错学习
    - Q-learning、策略更新
    - 探索-利用平衡
15. **社会学习系统** - 观察学习
    - 模仿、仿真、教学
    - 合作学习

#### 🎨 创意与创新（已实现）
16. **创造力系统** - 新想法生成
    - 盲目变异、选择性保留
    - 组合创新、类比推理
17. **问题解决系统** - 智能问题解决
    - 问题识别、方案生成
    - 启发式、分解、洞察

#### 🌍 环境与生态（已实现）
18. **资源竞争系统** - 4 类资源竞争
    - 物质、社会、信息、时间资源
    - 6 种竞争策略
19. **生态位分化系统** - 多样性增强
    - 专业化、共存
    - 生态位构建
20. **环境塑造系统** - Agents 改变世界
    - 物理、社会、文化、制度改造
    - 协同进化

### 🧠 分层记忆系统

#### 三层架构
- **工作记忆** - 容量 7±2，短期激活
- **短期记忆** - 容量 50，需要巩固
- **长期记忆** - 无限容量，缓慢衰减

#### 智能特性
- 🔄 自动巩固（短期 → 长期）
- 🔍 多层检索（优先级：工作 > 短期 > 长期）
- 🏷️ 5 种索引（重要性、情绪、来源、标签、时间）
- 🔗 语义链接（记忆间关联）

### 🎯 其他核心系统

- ⏰ **时间引擎** - 昼夜节律、活动模式
- 🤝 **群体记忆** - 集体记忆涌现、文化规范
- 🔗 **知识图谱** - GraphRAG、语义搜索
- 🎲 **推荐系统** - 智能内容推荐
- 🎨 **社交网络可视化** - Canvas 绘制
- 📈 **叙事时间线** - 情感曲线、事件回放

## 🚀 快速开始

### 前置要求

- Node.js 18+
- Docker & Docker Compose
- OpenAI API Key

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/zmzhace/world-slice.git
cd world-slice

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，添加你的 OPENAI_API_KEY

# 4. 启动 Qdrant 向量数据库
docker-compose up -d

# 5. 启动开发服务器
npm run dev

# 6. 打开浏览器
# 访问 http://localhost:3000
```

### 创建你的第一个世界

1. 点击"创建新世界"
2. 输入世界描述（例如："一个中世纪魔法学院"）
3. 等待盘古生成世界
4. 观察 Agents 自由互动
5. 见证故事自然涌现！

## 🎬 演示

### 涌现式叙事示例

```
Tick 1: Alice 和 Bob 初次相遇
  → 叙事识别器：检测到"初遇"模式

Tick 5: Alice 帮助 Bob 解决问题
  → 声誉系统：Alice 的善意 +0.05
  → 关系系统：Bob 对 Alice 的好感 +0.3

Tick 10: Charlie 嫉妒 Alice 和 Bob 的友谊
  → 叙事识别器：检测到"三角关系"模式
  → 戏剧张力系统：创建"冲突"张力

Tick 15: Charlie 散布关于 Alice 的谣言
  → 模因传播系统：负面模因开始传播
  → 声誉系统：Alice 的可信度 -0.15

Tick 20: Bob 发现真相，与 Charlie 对质
  → 故事弧检测器：识别到"真相揭露"故事弧
  → 戏剧张力系统：张力达到高潮

Tick 25: Alice、Bob、Charlie 达成和解
  → 叙事识别器：检测到"救赎"模式
  → 戏剧张力系统：张力释放
  → 叙事总结器：生成完整故事总结
```
